# Public Codex Skills

This repository is for Codex skills that are safe to publish publicly.

## Skills

- `skill-design-review`: Evaluates whether a Codex skill should be kept, simplified, merged, rewritten, deleted, or improved.
- `drawio-diagram-skill`: Generates editable draw.io / diagrams.net diagrams with export, validation, fallback links, and layout guidance. 中文关键词：流程图、架构图、画图 skill、汇报型流程图、系统架构图。
